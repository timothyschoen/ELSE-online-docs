[dir]

INLETs

  • 1st
    • bang - outputs <n>, <directory> and dumps files

OUTLETs

  • 1st
    • symbol - the files from the directory
  • 2nd
    • symbol - the current directory
  • 3rd
    • float - <n> number of found files
  • 4th
    • float - 1 if opened a new directory, 0 if it couldn't open

ARGUMENTs

  1. float - directory level, 0 — current patch's directory, 1 — parent's patch's, etc
  2. symbol - directory to open

FLAGs

  • - sets file extension type

    • METHODs

  • - opens a directory and loads contents
  • - reloads contents of the last opened directory
  • - resets to patch's directory
  • - outputs the current directory in the right outlet
  • - dump files from directory in the (second outlet)
  • - output number of found files in the (third outlet)
  • - seeks and output the file corresponding to that number
  • - increments seek value and outputs file
  • - sets extension
