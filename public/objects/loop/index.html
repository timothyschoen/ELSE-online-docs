[loop]

INLETs

  • 1st
    • float - sets number of iterations and starts the loop
    • list - sets range start/end and step (optionally) and run loop
    • bang - starts the loop
  • 2nd
    • float - sets number of iterations

OUTLETs

  • 1st
    • float/bang - counter output or bangs in bang mode

ARGUMENTs

  1. float - <f> — number of iterations, <f f f> — start, end, step
  2. float - sets the end value (optional)
  3. float - sets a counter step value

FLAGs

  • - sets offset value (default 0)
  • - sets counter step value (default 1)
  • - sets to bang mode

    • METHODs

  • - sets the starting value of the counter (for float)
  • - a float sets number of iterations, a list sets start/end and step (optionally)
  • - sets the increment value of the counter
  • - sets start/end values
  • - stops the loop
  • - continues the loop after being paused
