[xgate~]

INLETs

  • 1st
    • signal - input signal to route
    • float - selected outlet (0 is none)

OUTLETs

  • 1st
    • signal - routed output
  • 2nd
    • list - reports output number and off status

ARGUMENTs

  1. float - number of outlets/channels (min 1, max 500)
  2. float - crossfade time in ms
  3. float - initially selected channel

METHODs

  • - crossfade time in ms
