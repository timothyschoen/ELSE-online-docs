objects/sfont~.md

[sfont~]

INLETs

  • 1st
    • float - raw MIDI input

OUTLETs

  • 1st
    • signal - left output signal of stereo output
  • 2nd
    • anything - right output signal of stereo output
  • 3rd
    • anything - preset and scale name

ARGUMENTs

  1. symbol - soundfont file to load

FLAGs

  • - set to verbosity mode
  • - set the number of channels (default 16)
  • - set the gain, 0-1 (default 0.4)

    • METHODs

  • - loads soundfont file (.sf2/.sf3 extensions implied)
  • - key, velocity, channel
  • - same as note
  • - controller number, value, channel
  • - pitch bend value, channel
  • - value, channel
  • - key, value, channel
  • - program number, channel
  • - bank number, channel
  • - cents, channel
  • - pan value, channel
  • - sysex message
  • - resets synth and clears hanging notes
  • - prints version info on terminal
  • - prints soundfont info on terminal
  • - non-0 sets verbose mode
  • - list of 128 pitches in MIDI remaps all keys
  • - scale in cents to retune (12-tone temperament if no list)
  • - base MIDI pitch
  • - sets tuning <bank, program, channel & name> for a scale
  • - selects tuning <bank, program, channel> for a scale
  • - unselects tuning <bank, program, channel> for a scale
