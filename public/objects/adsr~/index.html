[adsr~]

INLETs

  • 1st
    • signal - gate value
    • float - gate value in MIDI velocity range (0-127 is 0-1)
    • bang - trigger/retrigger
  • 2nd
    • float/signal - attack time in ms
  • 3rd
    • float/signal - decay time in ms
  • 4th
    • float/signal - sustain amplitude (ratio to gate value)
  • 5th
    • float/signal - release time in ms

OUTLETs

  • 1st
    • signal - envelope signal
  • 2nd
    • float - envelope status (on=1 / off=0)

ARGUMENTs

  1. float - attack time in ms
  2. float - decay time in ms
  3. float - sustain amplitude (ratio to gate value)
  4. float - release time in ms

FLAGs

  • - sets to linear mode (default=log)

    • METHODs

  • - non-0 sets to "lin" mode, "log" otherwise
