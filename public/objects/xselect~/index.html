[xselect~]

INLETs

  • 1st
    • float - selected input (0 is none)
  • 2nd
    • signal - inputs to select from

OUTLETs

  • 1st
    • signal - selected inlet/channel
  • 2nd
    • list - reports input number and off status

ARGUMENTs

  1. float - number of inputs
  2. float - crossfade time in ms
  3. float - initially selected channel

METHODs

  • - crossfade time in ms
