[sequencer]

INLETs

  • 1st
    • bang - gets element(s) from sequence
    • float - sets index (from 1) and outputs it
  • 2nd
    • list - sets a new sequence
    • bang - sets an empty sequence

OUTLETs

  • 1st
    • float - element(s) from a sequence
  • 2nd
    • bang - when there is a rest
  • 3rd
    • float - bar number
  • 4th
    • bang - when reaching end of sequence

ARGUMENTs

  1. list - list of elements sets the sequence

METHODs

  • - goes to a position index in the sequence (from 1)
  • - set a new sequence
  • - clears sequence
