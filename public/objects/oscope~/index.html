    • METHODs

  • - sets number of samples per line (2-8192, default 256)
  • - sets number of lines in buffer (8-256, default 128)
  • - sets width/height (default 200 100)
  • - sets vertical range (default -1 1)
  • - sets front/signal RGB color (values 0-255)
  • - sets background RGB color (values 0-255)
  • - sets grid RGB color (values 0-255)
  • - <1> sets alternate drawing style (default 0)
  • - sets trigger mode; 0 (none, default), 1 (up) or 2 (down)
  • - sets threshold level for the trigger mode (default 0)
  • - onset time delay between displays (default 0)
  • - receive symbol (default empty)
