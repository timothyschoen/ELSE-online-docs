objects/drum.seq.md

[drum.seq]

INLETs

  • 1st
    • bang - gets sequence value and moves to the next
    • list - sets track, slot and status (no output)
    • float - sets slot value and output tracks values

OUTLETs

  • 1st
    • list - track and slot value (indexed from 1)
    • bang - when reaching the end of the sequence

FLAGs

  • - sets tracks (default 2)
  • - sets number of slots (default 8)
  • - sets cell size in pixels (default 20)
  • - sets embedding mode (default no embedding)

    • METHODs

  • - sets track values (1st value sets track number)
  • - sets slot value
  • - clears active cells (no output)
  • - export tracks, no float exports all as an array
  • - import contents as an array
  • - sets number of tracks (clears data)
  • - sets number of slots (clears data)
  • - sets cell size in pixels (clears data)
  • - non-0 save internal contents with the object
  • - array of contexts via the "export" message
