[player~]

INLETs

  • 1st
    • float - non-0 plays, <0> stops
    • bang - play (same as non-0)

OUTLETs

  • 1st
    • signal - the buffer signal of channel $nth
  • 2nd
    • bang - a bang when finishing playing the buffer

ARGUMENTs

  1. float - (optional) number of channels (max 64)
  2. symbol - the name of the file to open
  3. float - autostart <1; on, 0; off>
  4. float - loop <1; on, 0; off>

FLAGs

  • - (default 100)
  • - (default 0 1)
  • - (default 0)
  • - xfade mode on
  • - loop mode on

    • METHODs

  • - opens a file (no symbol opens dialog box) and plays
  • - sets a file to open (needs a reload message)
  • - shows the buffer subpatch window
  • - hides the buffer subpatch window
  • - sets start point in ms
  • - sets end point in ms
  • - sets start and end point proportionally (from 0 to 1)
  • - sets playing speed in %
  • - sets position (from 0 to 1) and starts playing it
  • - if no float is given, plays whole file. 1st float sets start, 2nd sets end (in ms) and 3rd sets speed rate
  • - stops playing and outputs 0 (cannot be resumed)
  • - pauses at a particular point (can be resumed)
  • - resumes playing after being paused
  • - sets fading time in ms (default 0)
  • - non-0 sets to crossfading mode (default 0)
  • - non-0 enables looping, <0> disables it (default 0)
