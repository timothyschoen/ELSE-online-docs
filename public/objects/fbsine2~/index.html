[fbsine2~]

INLETs

  • 1st
    • float/signal - frequency in Hz (negative values accepted)
    • list - 2 floats set x[n-1] and y[n-1] respectively

OUTLETs

  • 1st
    • signal - feedback sine chaotic signal

ARGUMENTs

  1. float - sets frequency in Hz
  2. float - sets 'im'
  3. float - sets 'fb'
  4. float - sets 'a'
  5. float - sets 'c'
  6. float - sets initial value of x[n]
  7. float - sets initial phase value of y[n]

METHODs

  • - sets values of 'im', 'fb', 'a' and 'c', respectively
  • - clears values of x[n-1] and y[n-1]
