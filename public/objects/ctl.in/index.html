objects/ctl.in.md

[ctl.in]

INLETs

  • 1st
    • float - raw MIDI data stream
  • 2nd
    • float - Controller number
  • 3rd
    • float - MIDI channel (0 for omni)

OUTLETs

  • 1st
    • float - control value
  • 2nd
    • float - control number
  • 3rd
    • float - MIDI channel

ARGUMENTs

  1. list - one value sets channel number. Two values set control number
