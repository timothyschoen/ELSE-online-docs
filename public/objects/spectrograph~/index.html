[spectrograph~]

INLETs

  • 1st
    • signal - incoming signal to graph
    • bang - graphs when rate is 0

FLAGs

  • - FFT size (default 1024, min 128)
  • - non-0 sets to dB amp scale (default linear)
  • - non-0 sets to log frequency scale (default linear)
  • - set horizontal/vertical dimensions (default 300 140)
  • - sets graph rate in ms (default 100)

    • METHODs

  • - sets FFT size
  • - non-0 sets to dB amp scale
  • - non-0 sets to log frequency scale
  • - set horizontal/vertical dimensions
  • - sets graph rate in ms
