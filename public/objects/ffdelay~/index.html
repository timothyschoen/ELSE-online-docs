objects/ffdelay~.md

[ffdelay~]

INLETs

  • 1st
    • signal - signal input into the delay line
  • 2nd
    • float/signal - delay time

OUTLETs

  • 1st
    • signal - the feed forward delayed signal

ARGUMENTs

  1. float - initial/maximum delay time

FLAGs

  • - delay size, defines the delay's maximum time (default 1000 ms or argument's value)
  • - sets delay time unit to "samples" (default is ms)

    • METHODs

  • - changes the delay size
  • - non-0 freezes, zero unfreezes
  • - clears the delay buffer
