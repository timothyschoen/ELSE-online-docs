[graph~]

INLETs

  • 1st
    • signal - incoming signal to graph

OUTLETs

  • 1st
    • signal - input signal comes out here

FLAGs

  • - buffer size in samples (default 441, minimum 64)
  • - buffer skip (default 10)
  • - background color in RGB (default 223 223 223)
  • - foreground color in RGB (default 0 0 0)
  • - set signal width (default 1)
  • - set lower and upper graph range (default -1 1)
  • - set horizontal/vertical dimensions (default 200 140)

    • METHODs

  • - buffer size in samples
  • - number of buffers to skip
  • - background color in RGB
  • - foreground color in RGB
  • - set signal width
  • - set min/max range
  • - set horizontal/vertical dimensions
