[eqdiv]

INLETs

  • 1st
    • bang - generate scale
    • float - set number of divisions and generate scale
  • 2nd
    • float - set interval ratio

OUTLETs

  • 1st
    • list - scale in cents or ratio

ARGUMENTs

  1. float - number of equal divisions
  2. float - interval ratio

FLAGs

  • - sets scale output to ratio instead of cents
