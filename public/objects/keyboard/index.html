objects/keyboard.md

[keyboard]

INLETs

  • 1st
    • float - MIDI Note value
    • list <f,f> - MIDI Note and Velocity values
  • 2nd
    • float - velocity of MIDI note value

OUTLETs

  • 1st
    • list <f,f> - MIDI Note and Velocity values

FLAGs

  • - width in pixels (default 17)
  • - height in pixels (default 80)
  • - number of octaves (default 4)
  • - number of lowest C (default 3 that is "C3")
  • - sets to toggle mode (default non toggle)
  • - velocity normalization value (default 0)
  • - sets send symbol (default 'empty')
  • - sets receive symbol (default 'empty')

    • METHODs

  • - list MIDI Notes to be on with velocity of 127
  • - list MIDI Notes to be off with velocity of 0
  • - set note (pitch/velocity) without output
  • - set key width
  • - set keyboard height
  • - transpose octaves up or down
  • - set number of octaves
  • - set number of lowest C (e.g. 4 = C4)
  • - set velocity normalization
  • - sets toggle mode on <1> of off <0>
  • - flushes hanging Note On keys
  • - sets send symbol
  • - sets receive symbol
