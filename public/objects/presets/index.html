[presets]

INLETs

  • 1st
    • float - sets a preset number and loads it
    • bang - save preset (same as 'save' without a float)
    • list - import presets from a list
  • 2nd
    • float - interpolation value (when in interpolation mode)

OUTLETs

  • 1st
    • list - presets as a list when receiving "export" message
  • 2nd
    • bang - when the preset is empty

ARGUMENTs

  1. list - one or more receive names
