[rescale]

INLETs

  • 1st
    • float/list - original value(s)
    • bang - outputs the last rescaled float value
  • 2nd
    • float - range parameters

OUTLETs

  • 1st
    • float/list - the rescaled value(s)

ARGUMENTs

  1. list - <min out, max out, exp factor> or <min in, max in, min out, max out, exp factor>

FLAGs

  • - sets clipping on

    • METHODs

  • - sets the next value to be rescaled via bang
  • - sets the exponential factor
  • - 1 clipping in, 0 off
