[pan4~]

INLETs

  • 1st
    • signal - signal input
  • 2nd
    • float/signal - X pan value; from -1 (Left) to 1 (Right)
  • 3rd
    • float/signal - Y pan value; from -1 (Back) to 1 (Front)

OUTLETs

  • 1st
    • signal - Left-Back channel
  • 2nd
    • signal - Left-Front channel
  • 3rd
    • signal - Left-Back channel
  • 4th
    • signal - Right-Back channel

ARGUMENTs

  1. float - initial X pan value
  2. float - initial Y pan value
