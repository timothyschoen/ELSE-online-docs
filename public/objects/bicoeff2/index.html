[bicoeff2]

INLETs

  • 1st
    • float - sets cutoff/center frequency
    • bang - generates coefficients
    • list <f, f, f> - sets frequency, Q/Slope and gain, and then outputs coefficients
    • anything - sets filter type <allpass, lowpass, highpass, bandpass, resonant, bandstop, eq, lowshelf, highshelf, off>. It takes 3 more optional arguments that set frequency, Q/Slope and gain
  • 2nd
    • float - sets "Q" or "Slope" and outputs coefficients
  • 3rd
    • float - sets gain in dB

OUTLETs

  • 1st
    • list - 5 coefficients for the vanilla [biquad~] object

ARGUMENTs

  1. symbol - (optional) sets type <highpass/etc>, (default='off')
  2. float - sets cutoff/center frequency (default=0)
  3. float - sets Q/slope (default=1)
  4. float - sets gain in dB (default=0)

METHODs

  • - sets "Q" or "Slope" and outputs coefficients
  • - sets gain in dB and outputs coefficients
