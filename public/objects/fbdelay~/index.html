[fbdelay~]

INLETs

  • 1st
    • signal - signal input to the delay line
  • 2nd
    • float/signal - delay time (im ms)
  • 3rd
    • float/signal - decay time (in ms)

OUTLETs

  • 1st
    • signal - the filtered/delayed signal

ARGUMENTs

  1. float - delay time
  2. float - feedback amount
  3. float - feedback mode; <0> ("t60") or <non-0> (gain mode)

FLAGs

  • - sets delay size (default 1000 ms or first argument's value if given)
  • - sets delay time unit to "samples" (default is ms)
  • - sets feedback mode to gain

    • METHODs

  • - changes the maximum delay size (in ms)
  • - non-0 freezes, zero unfreezes
  • - clears the delay buffer
  • - non-0 sets to gain mode
