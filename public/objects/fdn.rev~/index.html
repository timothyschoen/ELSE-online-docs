objects/fdn.rev~.md

[fdn.rev~]

INLETs

  • 1st
    • signal - input signal to reverberate
    • list - sets reflection times in ms
  • 2nd
    • float - decay time in seconds (t60)
  • 3rd
    • float - high frequency damping in % (from 0 to 100)

OUTLETs

  • 1st
    • signal - left output of the FDN reverberator
  • 2nd
    • signal - right output of the FDN reverberator

FLAGs

  • - t60 reverberation time in seconds (default 4)
  • - high frequency damping in % (default 0)

    • METHODs

  • - reverberation decay time in seconds (t60)
  • - high frequency damping in % (from 0 to 100)
  • - set number of lines and min/max times
  • - non-0 sets delay times exponentially
  • - clears the delay lines and reverberation
  • - print delay lines and other parameters in Pd's window
