[rec]

INLETs

  • 1st
    • play <list> - plays all tracks or tracks from the given list
    • speed <float> - sets playback speed in %
    • record <list> - records all tracks or tracks from the given list
    • stop <list> - stops (rec/play) all tracks or tracks from the given list
    • mute <list> - mutes all tracks or tracks from the given list
    • unmute <list> - unmutes all tracks or tracks from the given list
    • clear <list> - clears all tracks or tracks from the given list
    • open <symbol> - opens a text file
    • save <symbol> - saves to a text file
  • 2nd
    • anything - any message to be recorded in that inlet/track

OUTLETs

  • 1st
    • anything - the reversed message/list

ARGUMENTs

  1. float - number of tracks
  2. symbol - .txt file to open
