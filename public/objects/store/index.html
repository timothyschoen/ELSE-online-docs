[store]

INLETs

  • 1st
    • bang - outputs element from sequence and goes forward
    • float - sets index and outputs stored messages at index
  • 2nd
    • anything - messages to be stored in a sequence

OUTLETs

  • 1st
    • anything - the stored message
  • 2nd
    • anything - bang at the end of sequence or 'n' number of messages

ARGUMENTs

  1. float - float non-0 sets to store elements with the patch

METHODs

  • - sets index to output on next bang message
  • - outputs all stored messages sequentially
  • - clears all messages
  • - outputs number of elements in the right outlet
  • - opens [text] window
  • - closes [text] window
  • - clicking on the object opens [store]'s window
