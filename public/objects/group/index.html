[group]

INLETs

  • 1st
    • anything - input messages to be regrouped
    • bang - outputs the remainder
  • 2nd
    • float - calculate or output the last calculated value

OUTLETs

  • 1st
    • anything - the regrouped message
  • 2nd
    • bang - when there's no remainder (group is empty)

ARGUMENTs

  1. float - group size

FLAGs

  • - trims selectors (list/symbol) on the output

    • METHODs

  • - clears the remainder
