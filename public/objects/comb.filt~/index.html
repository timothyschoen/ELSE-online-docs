[comb.filt~]

INLETs

  • 1st
    • signal - signal input to the filter
  • 2nd
    • float/signal - frequency in Hz
  • 3rd
    • float/signal - decay coefficient

OUTLETs

  • 1st
    • signal - output from comb filter

ARGUMENTs

  1. float - maximum and initial delay time in ms
  2. float - decay coefficient
  3. float - decay mode <0> or gain mode <non-0>

FLAGs

  • - sets feedback mode to gain

    • METHODs

  • - clears the delay buffer
  • - non-0 sets to gain mode instead of 't60'
