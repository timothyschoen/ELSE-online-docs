[rec.file~]

INLETs

  • 1st
    • start - start recording
    • stop - stop recording
    • float - non-0 starts playing, 0 stops
  • 2nd
    • signal - signal inputs to record for each channel

OUTLETs

  • 1st
    • bang - outputs a bang when stops recording

ARGUMENTs

  1. symbol - (optional) name of the file to save to
  2. float - number of channels to record

METHODs

  • - sets file name (no symbol opens dialog box)
  • - sets sample rate
  • - sets fade time (minimum 1)
  • - sets maximum recording in ms
