[pluck~]

INLETs

  • 1st
    • signal - trigger (determines the amplitude)
    • float - non-0 triggers and sets amplitude
    • bang - triggers with last control amplitude (default 1)
  • 2nd
    • float/signal - frequency in Hz (minimum 1)
  • 3rd
    • float/signal - decay time in ms
  • 4th
    • float/signal - filter cutoff frequency
  • 5th
    • float/signal - optional noise input (with the -in flag)

OUTLETs

  • 1st
    • signal - the Karplus-Strong output

ARGUMENTs

  1. float - frequency in Hz
  2. float - decay time in ms
  3. float - filter cutoff frequency

FLAGs

  • - creates an extra right inlet for noise input
