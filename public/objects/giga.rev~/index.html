objects/giga.rev~.md

[giga.rev~]

INLETs

  • 1st
    • signal - signal input (mono in, stereo out)

OUTLETs

  • 1st
    • signal - left output channel of the reverb
  • 2nd
    • signal - right output channel of the reverb

FLAGs

  • - maximum room size in square meters (default 300)
  • - room size in square meters (default 50)
  • - decay time in seconds (default 7)
  • - high frequency damping (default 0.5)
  • - stereo spread (default 0.5)
  • - input bandwidth (default 0.5)
  • - dry level (default 0.5)
  • - early reflections level (default 0.25)
  • - late reflections level (default 0.25)
  • - wet level (default 1)

    • METHODs

  • - room size in square meters
  • - high frequency damping (0-1)
  • - decay time in seconds (0.001-3600)
  • - input bandwidth (0-1)
  • - stereo spreading (0-1)
  • - early reflections gain level (0-1)
  • - late reflections gain level (0-1)
  • - dry gain level (0-1)
  • - wet gain level (0-1) overrides early and late
  • - clears delay lines
  • - print parameters on Pd Window
