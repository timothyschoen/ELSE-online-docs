[sample~]

INLETs

  • 1st
    • open <symbol> - opens sample file
    • info - output size, sample rate and channels
    • reopen - reopens the previously opened file
    • clear - fills all arrays with zeros
    • trim <f,f> - trims to a sample range
    • save <symbol> - sets file to write to
    • depth <f> - bit depth (16, 24 or 32)
    • show - opens buffer supbatch window
    • hide - closes buffer supbatch window
    • sr <f> - sets sample rate (default patch's)
    • rename <symbol> - renames buffer
    • normalize <f> - normalizes to the given float value
    • size <f> - maximum buffer size in samples
    • ms <f> - maximum buffer size in ms

OUTLETs

  • 1st
    • list <f,f,f> - sample's size, sample rate and channels

ARGUMENTs

  1. symbol - buffer name
  2. symbol - audio file to load (optional)

FLAGs

  • - maximum buffer size in samples
  • - maximum buffer size in ms
  • - number of channels (default 1)
