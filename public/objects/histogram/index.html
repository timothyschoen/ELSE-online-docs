[histogram]

INLETs

  • 1st
    • float - positive integer to record into histogram
    • list - note on pitch to record into histogram

OUTLETs

  • 1st
    • list - occurrences and index after input
  • 2nd
    • list - table values at 'export' message
  • 3rd
    • float - number of occurrences at 'query' message

ARGUMENTs

  1. symbol - set table name

FLAGs

  • - sets table size (default 128)
  • - embeds histogram data with the patch (default don't)

    • METHODs

  • - clears histogram table
  • - query number of occurrences of an index
  • - export histogram table as a list
  • - import histogram data as a list
  • - non-0 embeds histogram data with the patch
  • - clears and resizes table
  • - sets table name
