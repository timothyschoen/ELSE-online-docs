[grain.live~]

INLETs

  • 1st
    • signal - signal input to be granulated
    • bang - plays an event of cloud grains
  • 2nd
    • float - number of grains in the event (default 16, min 1, max 256)

OUTLETs

  • 1st
    • signal - left channel output
  • 2nd
    • signal - right channel output

    • METHODs

  • - sets cloud event duration in ms (default 500)
  • - non-0 sets to synchronous mode (default 0)
  • - sets min/max grain sizes in ms (default 50 to 450)
  • - sets min/max transposition in cents (default -1200 to 1200)
  • - sets left/right pan boundaries (default -1 to 1)
  • - sets min/max position in delay line (default 0 to 1)
  • - sets min/max amplitude values (default 0.1 to 1)
  • - sets probability in % of grain being reversed (default 0)
  • - envelope type (sin, hann, tri, gauss) or function list
  • - non-0 autotunes to a given scale (default 0)
  • - scale to autotune to in cents (default equal temperament)
