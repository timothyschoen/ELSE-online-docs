[plaits~]

INLETs

  • 1st
    • float/signal - pitch input
    • bang - control trigger (when in trigger mode)
  • 2nd
    • signal - signal trigger input
  • 3rd
    • signal - level for built-in VCA or excitation signal

OUTLETs

  • 1st
    • signal - regular signal output
  • 2nd
    • signal - secondary (auxiliary) signal output
  • 3rd
    • anything - output info on setting model or 'dump'

ARGUMENTs

  1. float - pitch
  2. float - harmonics
  3. float - timbre
  4. float - morph
  5. float - cutoff
  6. float - decay

FLAGs

  • - set to pitch input in CV, MIDI, or v/oct (default Hz)
  • - set model number (default 0)
  • - set to trigger mode (default regular)

    • METHODs

  • - set frequency mode to Hz
  • - set frequency mode to MIDI
  • - set frequency mode to CV
  • - set frequency mode to v/oct
  • - non-0 sets to trigger mode
  • - set model number (0-15)
  • - set harmonics (0-1)
  • - set timbre (0-1)
  • - set morph (0-1)
  • - set lowpass gate cutoff/color (0-1)
  • - set lowpass gate decay (0-1)
  • - output information on terminal window
  • - output information on rightmost outlet
