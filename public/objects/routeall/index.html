[routeall]

INLETs

  • 1st
    • anything - any message to be routed completely
  • 2nd
    • float - set element number to match (default 0)

OUTLETs

  • 1st
    • anything - if an input message matches an argument, the corresponding outlet sends that message
  • 2nd
    • anything - any unmatched message

ARGUMENTs

  1. anything - arguments to match the 1st element of a message
