[envgen~]

INLETs

  • 1st
    • float/signal - gate; on/attack (non-0) or off/release (0)
    • bang - attacks the envelope with the last gate on value
    • list - sets and runs envelope (pairs of duration & target. if odd, 1st float is starting point)
  • 2nd
    • list - sets the envelope and doesn't run it

OUTLETs

  • 1st
    • signal - envelope signal
  • 2nd
    • float - status; 1 <on>, 0 <off>
