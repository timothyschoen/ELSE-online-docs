[ramp~]

INLETs

  • 1st
    • signal - signal trigger (starts ramp from the reset point value)
    • bang - control rate trigger (starts ramp from the reset point value)
  • 2nd
    • float/signal - sets the increment value
  • 3rd
    • float/signal - sets the minimum value
  • 4th
    • float/signal - sets the maximum value

OUTLETs

  • 1st
    • signal - the ramp signal
  • 2nd
    • bang - a bang when ramp is finished

ARGUMENTs

  1. float - increment value
  2. float - minimum value
  3. float - maximum value
  4. float - start/reset value

FLAGs

  • - sets the reset point value
  • - 0 wrap, 1 clip, 2 reset
  • - stops incrementing
  • - (re)starts incrementing
  • - stops and goes back to the reset point

    • METHODs

  • - 0 wrap, 1 clip, 2 reset
  • - prevents the ramp from automatically starting
