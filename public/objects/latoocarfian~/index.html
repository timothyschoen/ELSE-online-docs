INLETs

  • 1st
    • float/signal - frequency in Hz (negative values accepted)
    • list - 2 floats set x[n-1] and y[n-1], respectively

OUTLETs

  • 1st
    • signal - Latoocarfian chaotic signal

ARGUMENTs

  1. float - sets frequency in Hz
  2. float - sets a
  3. float - sets b
  4. float - sets c
  5. float - sets d
  6. float - sets initial value of y[n-1]
  7. float - sets initial value of x[n-1]

METHODs

  • - list sets values of 'a', 'b', 'c' and 'd'
