[tabplayer~]

INLETs

  • 1st
    • float - non-0 plays, 0 stops
    • bang - play (same as non-0)
    • signal - gate on/impulse — starts, gate off — stops (not in trigger mode)

OUTLETs

  • 1st
    • signal - the playback of a channel
  • 2nd
    • bang - when it stops/finishes playing or when looping

ARGUMENTs

  1. symbol - table name (optional)
  2. float - number of output channels (maximum 64)

FLAGs

  • -
  • -
  • - sets to trigger mode
  • - sets to loop mode
  • - sets to trigger mode
  • - sets to loop mode
  • - sets to crossfade mode
  • -
  • -

    • METHODs

  • - non-0 sets to trigger mode, zero sets to gate mode
  • - sets array name
  • - sets start point in ms
  • - sets end point in ms
  • - sets start and end point range proportionally (from 0 to 1)
  • - sets position proportionally within range (from 0 to 1)
  • - resets range from 0 to array size
  • - sets playing speed in %
  • - start playing, optional <start in ms, end in ms, speed rate>
  • - stops playing and outputs 0 (cannot be resumed)
  • - pauses at a particular point (can be resumed)
  • - resumes playing after being paused
  • - non-0 enables looping, <0> disables it
  • - sets fade time in ms
  • - sets to crossfade mode when looping
  • - sets sample rate of sample (default, Pd's sample rate)
