[metronome]

INLETs

  • 1st
    • bang - start or restart metronome
    • float - non-0 (re)starts, zero stops
  • 2nd
    • float - set tempo value in BPM

OUTLETs

  • 1st
    • bang - bang at every beat
  • 2nd
    • list - bar, sub-bar, beat, sub-beat count
  • 3rd
    • float - beat phase (0-1)
  • 4th
    • list - actual beat length, actual tempo and bar duration in ms

ARGUMENTs

  1. float - tempo in BPM

FLAGs

  • - set a clock name
  • - set a beat length
  • - sets to subdivision (subtempo) mode

    • METHODs

  • - start or restart metronome
  • - stop metronome
  • - pause metronome
  • - continue if paused
  • - set a beat length
  • - set time signature symbol and group value
  • - sets to subdivision (subtempo) mode
  • - set tempo value in BPM
