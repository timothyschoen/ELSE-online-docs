[downsample~]

INLETs

  • 1st
    • signal - signal to be downsampled
    • bang - restart cycle and sync output
  • 2nd
    • float/signal - rate (in Hz) used to downsample the input signal

OUTLETs

  • 1st
    • signal - downsampled signal

ARGUMENTs

  1. float - rate in Hz
  2. float - interpolation 0 (off) or 1 (on)

METHODs

  • - interpolation 0 (off, default) or 1 (on)
