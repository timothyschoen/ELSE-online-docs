[oscbank2~]

INLETs

  • 1st
    • freq <list> - list of frequencies for all oscillators in the bank
    • amp <list> - list of amplitudes for all oscillators in the bank
    • phase <list> - list of phases (0-1) for all oscillators in the bank
    • ramp <list> - list of ramp time for all oscillators in the bank
    • rampall <float> - sets ramp time for all oscillators

OUTLETs

  • 1st
    • signal - the oscillator bank output

ARGUMENTs

  1. float - number of oscillators
  2. float - ramp time in ms

FLAGs

  • - sets list of frequencies for all oscillators
  • - sets list of amplitudes for all oscillators
  • - sets list of phases for all oscillators
  • - sets ramp time for all oscillators
  • - sets a ramp time for all oscillators (default all 10)

    • METHODs

  • - list of frequencies for all oscillators in the bank
  • - list of amplitudes for all oscillators in the bank
  • - list of phases (0-1) for all oscillators in the bank
  • - list of ramp time for all oscillators in the bank
  • - sets ramp time for all oscillators
