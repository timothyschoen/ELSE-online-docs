[play.file~]

INLETs

  • 1st
    • bang - (re)start playing the sample
    • float - non-0 (re)starts playing, zero stops

OUTLETs

  • 1st
    • signal - the sample signal of channel $nth
  • 2nd
    • bang - a bang when finishing playing the sample

ARGUMENTs

  1. float - (optional) number of channels (max 64)
  2. symbol - the name of the file to open
  3. float - autostart <1; on, 0; off>
  4. float - loop <1; on, 0; off>

FLAGs

  • - turns loop mode on

    • METHODs

  • - (re)start playing the sample
  • - stop playing the sample
  • - opens a file with the symbol name (no symbol opens dialog box) and starts playing
  • - sets a file to open next time it starts playing
  • - loop on <1> or off <0>
