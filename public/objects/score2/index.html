[score2]

INLETs

  • 1st
    • start - starts the sequence
    • stop - stops the sequence
    • float - non-0 starts, 0 stops sequence

OUTLETs

  • 1st
    • anything - event data
  • 2nd
    • float - event duration in ms
  • 3rd
    • anything - score data (bar number, tempo and other things by the user)
  • 4th
    • bang - when sequence is over

ARGUMENTs

  1. symbol - score file to load

METHODs

  • - pauses playing
  • - continues playing
  • - opens edit window
  • - closes edit window
  • - import score from a file, no symbol opens dialog window
  • - write score to file, no symbol opens dialog window
