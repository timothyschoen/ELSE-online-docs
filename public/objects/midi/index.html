[midi]

INLETs

  • 1st
    • float - raw MIDI data to record
    • bang - external clock tick

OUTLETs

  • 1st
    • float - raw MIDI data stream from a MIDI file
  • 2nd
    • bang - sent at the end of a sequence

ARGUMENTs

  1. symbol - a MIDI or text file to load

  • - sets the loop mode

  • - stops recording/playing and goes back to start
  • - pauses recording/playing
  • - continues recording/playing
  • - sets a reading speed in % of original
  • - outputs the MIDI data stream at once
  • - flushes hanging notes
  • - clears sequence from the object
  • - saves to a MIDI file
