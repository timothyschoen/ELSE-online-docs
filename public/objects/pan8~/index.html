[pan8~]

INLETs

  • 1st
    • signal - signal input
  • 2nd
    • float/signal - X horizontal value; from -1 (Left) to 1 (Right)
  • 3rd
    • float/signal - Y vertical value; from -1 (Back) to 1 (Front)
  • 4th
    • float - spread parameter (from -1 to 1)

OUTLETs

  • 1st
    • signal - channel output 1
  • 2nd
    • signal - channel output 2
  • 3rd
    • signal - channel output 3
  • 4th
    • signal - channel output 4
  • 5th
    • signal - channel output 5
  • 6th
    • signal - channel output 6
  • 7th
    • signal - channel output 7
  • 8th
    • signal - channel output 8

FLAGs

  • - sets spread parameter (default 0)
  • - sets mode <0> or <1> (default 0)
