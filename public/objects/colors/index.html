[colors]

INLETs

  • 1st
    • bang - output/convert color

OUTLETs

  • 1st
    • anything - color values (in RGB, hex, iemgui or ds)

FLAGs

  • - sets to RGB list output (default hexadecimal)
  • - sets to iemguis output (default hexadecimal)
  • - sets to Data Structures output (default hexadecimal)

    • METHODs

  • - open color picker
  • - RGB (Red, Green, Blue) color values
  • - hexadecimal color value
  • - iemgui's color format value
  • - Data Structures' color value
  • - CMYK (Cyan, Magenta, Yellow, and Key) color values
  • - HSL (Hue, Saturation, and Lightness) color values
  • - HSV (Hue, Saturation, and Value) color values
  • - grayscale value from 0 to 100
  • - set conversion to "rgb," "hex," "gui" or "ds
