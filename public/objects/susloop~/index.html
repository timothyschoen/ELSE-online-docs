[susloop~]

INLETs

  • 1st
    • float/signal - 0 to non-0; starts and loops. Non-0 to 0; loop is off
    • bang - trigger; starts from 0 and loops
  • 2nd
    • float/signal - sets increment value (rate/speed)

OUTLETs

  • 1st
    • signal - the control signal for the sampler
  • 2nd
    • float - status (on; 1 / off; 0)

ARGUMENTs

  1. float - increment value
  2. float - loop start
  3. float - loop end
  4. float - sample size

METHODs

  • - loop is off
  • - stops and goes back to 0, expects a trigger to restart
  • - pauses/stops incrementing
  • - continues after being paused
  • - sets sustain loop range in samples <start, end>
  • - sets sample size
