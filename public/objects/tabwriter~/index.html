[tabwriter~]

INLETs

  • 1st
    • float/signals - non-0 starts recording, 0 stops it
    • rec - (re)starts recording
    • stop - stops recording
  • 2nd
    • signal - signal to record into an array channel 'n'

OUTLETs

  • 1st
    • signal - output index we're recording into
  • 2nd
    • bang - when recording reaches the end of the array

ARGUMENTs

  1. symbol - array name (optional)
  2. float - channels to record (max 64)

FLAGs

  • -
  • -
  • -
  • -
  • -

    • METHODs

  • - sets array for recording signals
  • - start point in ms (default 0)
  • - end point in ms (default array's initial size)
  • - sets start and end point range proportionally (from 0 to 1)
  • - resets start/end from 0 to (current) array's size
  • - non-0 continue recording from where it last stopped
  • - non-0 enables loop recording, 0 disables it
