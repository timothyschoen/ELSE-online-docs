[lfo]

INLETs

  • 1st
    • float - frequency in Hz up to 100 (negative values accepted)
    • anything - sets waveform (sine, tri, saw, vsaw, and square)
  • 2nd
    • float - phase reset from 0 1
  • 3rd
    • float - refresh rate in ms (minimum is 5 ms)

OUTLETs

  • 1st
    • float - LFO's waveform in the range from 0 127

ARGUMENTs

  1. float - sets frequency in Hz (default 0)
  2. float - sets initial phase from 0 1 (default 0)
  3. float - sets refresh rate in ms (default 5)

FLAGs

  • - sets waveform to sine (default)
  • - sets waveform to triangular
  • - sets waveform to sawtooth
  • - sets waveform to square and width (from 0 to 1)
  • - sets waveform to variable sawtooth and width (from -1 to 1)
