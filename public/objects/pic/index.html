objects/pic.md

[pic]

INLETs

  • 1st
    • open <symbol> - open an image file named by the symbol

OUTLETs

  • 1st
    • bang - a bang is output if you click on the image
    • float - 1/0 when in latch mode
    • list - pic width/height when in report size mode

FLAGs

  • - sets file name to open (default 'empty')
  • - sets send symbol (default 'empty')
  • - sets receive symbol (default 'empty')
  • - sets to outline mode (default no outline)
  • - sets to "report size" mode (default no report)
  • - sets to "latch" mode (default "bang" mode)

    • METHODs

  • - open an image file named by the symbol
  • - same as open, but pd doesn't ask to save changes
  • - non-0 sets to latch mode
  • - non-0 sets to outline mode
  • - non-0 sets to report size mode
  • - sets a send symbol
  • - sets a receive symbol
