[highpass~]

INLETs

  • 1st
    • signal - signal to be filtered
  • 2nd
    • float/signal - central frequency in Hz
  • 3rd
    • float/signal - filter resonance (Q or bandwidth)

OUTLETs

  • 1st
    • signal - filtered signal

ARGUMENTs

  1. float - central frequency in Hz
  2. float - resonance, either in 'Q' or 'bw'

FLAGs

  • - sets resonance parameter to bandwidth in octaves

    • METHODs

  • - clears filter's memory if you blow it up
  • - 1 (bypasses input signal) or 0 (doesn't bypass)
  • - sets resonance parameter to bandwidth in octaves
  • - sets resonance parameter to Q (default)
