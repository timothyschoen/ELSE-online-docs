[grain.synth~]

INLETs

  • 1st
    • bang - plays <dur> of pitched clouds
  • 2nd
    • float - density (number of grains)

OUTLETs

  • 1st
    • signal - left channel output
  • 2nd
    • signal - right channel output

    • METHODs

  • - number of grains in event cloud (default 16, min 1, max 256)
  • - sets table name with waveform
  • - non-0 sets to synchronous mode (default 0)
  • - sets cloud event duration in ms (default 500)
  • - sets min/max grain sizes in ms (default 50 to 450)
  • - sets min/max pitch values in MIDI (default 30 to 90)
  • - sets left and right pan boundaries (default -1 to 1)
  • - min/max amplitude values (default 0.1 to 1)
  • - envelope type (sin, hann, tri, gauss) or function list
  • - non-0 autotunes to a given scale (default 0)
  • - scale to autotune to in cents (default equal temperament)
  • - sets scale's base MIDI pitch (default 60)
